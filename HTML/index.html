¿Que es la Programación Competitiva?

La programación competitiva se define como un desafío mental, donde los participantes compiten entre sí para resolver un conjunto de problemas de programación en un tiempo determinado. Cada competición o torneo consta de un conjunto variable de problemas, con su número y duración sujetos a variación.

Torneos

Encontrarás distintos torneos en los que podrás participar virtualmente, cada torneo cuenta un set de problemas y un tiempo de duración, el cual deberás resolver la mayor cantidad de problemas en el menor tiempo posible.
