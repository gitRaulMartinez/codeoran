<p>La primera línea de entrada contiene un número entero, <strong>n</strong> (1 ≤ n ≤ 50), que indica el número de edificios. Cada una de las siguientes <strong>n</strong> líneas de entrada describe un edificio. Un edificio se describe mediante tres números enteros: <strong>s</strong> (1 ≤ s ≤ 98), que indica la esquina inferior izquierda del edificio, <strong>w</strong> (1 ≤ w ≤ 98), que indica el ancho del edificio, y <strong>h</strong> (1 ≤ h ≤ 100), indicando la altura del edificio. Suponga que <strong>s</strong> + <strong>w</strong> ≤ 99, es decir, ningún edificio terminará más allá de (99, 0).</p>
